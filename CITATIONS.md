# Complete Citations for FairSample

This document provides complete citations for all research papers implemented in FairSample.

## Citation Guidelines

When using FairSample in academic work:

1. **Cite FairSample itself** (see below)
2. **Cite the original papers** for techniques you use
---

## Citing FairSample

### BibTeX
```bibtex
@software{fairsample2026,
  author = {Uwaish, Mohd},
  title = {FairSample: Fair sampling for imbalanced datasets with complexity analysis},
  year = {2026},
  url = {https://github.com/mohdUwaish59/fairsample},
  note = {Python package version 0.1.0}
}
```

### APA Style

Uwaish, M. (2026). *FairSample: Fair sampling for imbalanced datasets with complexity analysis* (Version 0.1.0) [Computer software]. https://github.com/mohdUwaish59/fairsample
